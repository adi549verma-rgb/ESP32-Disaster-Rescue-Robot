# ESP32 Disaster Rescue Robot

## Overview

This project is an IoT-based disaster rescue robot developed using ESP32. The robot can be controlled through Bluetooth, detect obstacles using ultrasonic sensors, and identify possible human presence by monitoring CO₂ levels with the SCD40 sensor.

---

## Features

- Bluetooth controlled robot
- CO₂ detection
- Temperature monitoring
- Humidity monitoring
- Obstacle detection
- LED alert
- Buzzer alert

---

## Hardware

- ESP32
- SCD40
- HC-SR04 ×2
- L298N
- DC Motors
- LED
- Buzzer

---

## Bluetooth Commands

| Command | Action |
|----------|--------|
| F | Forward |
| B | Backward |
| L | Left |
| R | Right |
| S | Stop |
